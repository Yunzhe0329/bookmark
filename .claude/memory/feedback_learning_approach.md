name: Learning-oriented teaching approach
description: User wants step-by-step guidance with explanations, not code dumps
type: feedback
---

Do NOT write all code at once. Guide the user step by step, explaining what each action does and why before proceeding.

**Why:** This project is a learning exercise. The user wants to understand the concepts behind each step, not just have the code written for them.

**How to apply:** Before writing any code, explain what it is, why it's needed, and what it does. Ask for confirmation before moving to the next step. Teach concepts alongside implementation.
